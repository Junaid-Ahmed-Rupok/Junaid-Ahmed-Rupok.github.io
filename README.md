## **Installation & Setup**

### 1. Clone the repository:

```bash
git clone https://github.com/Junaid-Ahmed-Rupok/portfolio.git
cd portfolio
```

### 2. Local Development Server

```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have it installed)
npx serve .

# Using PHP
php -S localhost:8000
```

### 3. Open in browser

```bash
# Navigate to
http://localhost:8000
```
